# Argon - Combat HUD (The Witcher TRPG)

![Foundry VTT](https://img.shields.io/badge/Foundry%20VTT-v13-green)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

Integration of [Argon - Combat HUD (CORE)](https://foundryvtt.com/packages/enhancedcombathud) for the [The Witcher TRPG](https://github.com/witchertrpg-foundryvtt/TheWitcherTRPG) system in Foundry VTT.

Adds a fully featured combat HUD with action panels for weapons, magic, verbal combat, brawling, special attacks, and full-round actions — for both player characters and monsters.

![Demo](./docs/argonhudthewitcher.gif)

---

## Requirements

| Dependency | Notes |
|---|---|
| [Foundry VTT](https://foundryvtt.com) | v13 (not tested on v12) |
| [Argon - Combat HUD (CORE)](https://foundryvtt.com/packages/enhancedcombathud) | v4.0.7 (not tested on v14) |
| [The Witcher TRPG](https://github.com/witchertrpg-foundryvtt/TheWitcherTRPG) | v13.0.0 or higher |

## Installation

Install via Foundry VTT's module browser, or paste the manifest URL directly:

```
https://github.com/pedroaugustobt/enhancedcombathud-thewitchertrpg/releases/latest/download/module.json
```

## Features

### Action Panels

- **Weapons** — Quick access to all equipped weapons, organized in weapon sets
- **Magic** — Witcher Signs, Spells, Hexes, and Rituals displayed in categorized drawers
- **Verbal Combat** — Dedicated panel for verbal combat actions
- **Consumables** — Use items and consumables directly from the HUD
- **Skills** — Access character skills from the drawer panel

### HUD Elements

- **Portrait Panel** — Character portrait with HP, Stamina, Vigor, Adrenaline, and Luck stats
- **Movement HUD** — Visual movement tracker
- **Rest Button** — Trigger a Daily Rest from the HUD
- **Monster Support** — Works for both `character` and `monster` actor types
